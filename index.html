домашнее задание(B6.13)

Файлы программы:

  • Модуль album.py
  • Модуль сервера album_server.py
  • База данных albums.sqlite3
  • Все файлы в архиве тут

    • Краткое описание

    Файлы должны быть в одной директории.

    Веб сервер принимает GET запросы по адресу /albums/<artist> и выводит на экран сообщение с количеством альбомов исполнителя artist и списком названий этих альбомов. например http://127.0.0.1:8080/albums/Kino

    Веб сервер принимает POST запросы по адресу /albums/, пример запроса http -f POST http://localhost:8080/albums artist="New Artist" year="1999" genre="Rock" album="Super", при передаче неверных или пустых данных возвращает ошибку с описанием, примеры на скринах, в коде есть пояснения
